Bar Charts

There are two types of bar charts. Two measure bar charts (gradient bar charts) and Two Dimension Bar Charts (stacked bar charts).

Two Measures

This chart looks at flights and counts the number of aircraft owned by each carrier. It also, using a gradient, shows the number of flights made per plane.

query: table('malloy-data.faa.flights') -> {
  nest: by_carrier is {
    group_by: carrier
    aggregate: aircraft_count is count(distinct tail_num)
    aggregate: flights_per_aircraft is count() / count(distinct tail_num)
  }
}

WITH __stage0 AS (
  SELECT
    group_set,
    CASE WHEN group_set=1 THEN base.carrier END as carrier__1,
    CASE WHEN group_set=1 THEN count(distinct base.tail_num) END as aircraft_count__1,
    CASE WHEN group_set=1 THEN COUNT( 1)/count(distinct base.tail_num) END as flights_per_aircraft__1
  FROM `malloy-data.faa.flights` as base
  CROSS JOIN (SELECT row_number() OVER() -1  group_set FROM UNNEST(GENERATE_ARRAY(0,1,1)))
  GROUP BY 1,2
)
SELECT
  ARRAY_AGG(CASE WHEN group_set=1 THEN STRUCT(carrier__1 as carrier, aircraft_count__1 as aircraft_count, flights_per_aircraft__1 as flights_per_aircraft) END IGNORE NULLS  ORDER BY  aircraft_count__1 desc ) as by_carrier
FROM __stage0

Two Dimensions

In this case we are going to look at carriers by flight count and stack the destination. We are only going to look at flights with the destination SFO, OAK or SJC.

query: table('malloy-data.faa.flights') -> {
  where: destination: 'SFO' | 'OAK' | 'SJC'
  top: 10
  nest: by_carrier is {
    group_by: carrier
    aggregate: flight_count is count()
    group_by: destination
  }
}
